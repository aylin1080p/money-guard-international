# Auth Assets

These SVG files were renamed from the Figma export archive using a stable pattern:

- section first: `auth`
- visual role next: `wave-fill`, `wave-stroke`, `card-fill`
- source size after that: for example `480x167`
- stable content suffix last: short hash derived from the original SVG file

Examples:

- `auth-wave-fill-480x167-0dc439.svg`
- `auth-wave-stroke-337x72-49f923.svg`
- `auth-card-fill-336x88-cdd3e7.svg`

This avoids names like `Vector 7-3.svg` while keeping each export traceable.
The short hash suffix is intentional because several exports share the same role and size but still differ in shape or gradient positioning.
